Utilities

The core of the Joose “library” is solving some must-have or basic problems, such as class handling.

Functionality in Joose Core

  • Class Handling

    • Checking for a class to an element joose.utils.hasClass(document.body, 'myClass');
    • Adding a class to an element joose.utils.addClass(document.body, 'myClass');
    • Removing a class from an element joose.utils.removeClass(document.body, 'myClass');
    • Toggling a class on an element joose.utils.toggleClass(document.body, 'myClass');
